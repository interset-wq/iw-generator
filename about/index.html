    About Me

    This is a personal blog powered by iw-generator.

    What is iw-generator?

    iw-generator is a general-purpose Markdown static site generator library that supports:

    • Blog mode: Full-featured blog with categories, tags, and archives
    • Docs mode: Documentation site similar to MkDocs Material
    • Multiple themes: iw (GitHub Issues style), blog (Gmeek-like), doc (Material-like)

    Features

    • Markdown rendering with syntax highlighting
    • Tag and category support
    • RSS feed generation
    • Search functionality
    • Dark/light theme toggle
    • Mobile-responsive design
