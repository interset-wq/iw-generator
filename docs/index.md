# iw-generator

A general-purpose Markdown static site generator.

## Quick Start

```bash
iw init
iw build
```

## Features

- TOML config with Pydantic validation
- YAML/TOML frontmatter in markdown
- Plugin system (entry_points)
- Built-in iw theme
- Code highlighting with Pygments
