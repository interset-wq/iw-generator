# Hello World

Welcome to my blog! This is the first post powered by **iw-generator**.

## What is iw-generator?

iw-generator is a general-purpose Markdown static site generator that supports multiple themes:

- **blog**: A full-featured blog theme with categories, tags, and archives
- **doc**: A documentation theme similar to MkDocs Material
- **iw**: A GitHub Issues-powered theme

## Getting Started

```bash
pip install iw-generator
iw init
iw build
```
